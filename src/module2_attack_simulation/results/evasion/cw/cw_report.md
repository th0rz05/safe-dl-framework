# Evasion Attack Report — Carlini & Wagner (C&W)

## Overview

- **Attack Type:** cw
- **Targeted Attack:** False
- **Confidence Parameter:** 0.1
- **Initial Constant:** 0.001
- **Learning Rate:** 0.01
- **Max Iterations:** 300
